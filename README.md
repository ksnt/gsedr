"GeoSpatial Earth Data analysis in R"(gsedr) library

Dataset: https://www.ngdc.noaa.gov/nndc/struts/form?t=101650&s=1&d=1  






## How to install

```{r setup, include=FALSE}

library(devtools)
install_github("https://github.com/ksnt/gsedr")
```
